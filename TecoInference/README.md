# TecoInferenceEngine

## 介绍

TecoInferenceEngine（小模型）是基于太初加速卡特性研发的高性能推理加速引擎，基于AI编译技术，提供高效的小模型推理解决方案，并针对分类、检测、分割、自然语言处理、语音等场景常用的各类经典和前沿的AI模型提供适配支持。

## 模型适配全流程
