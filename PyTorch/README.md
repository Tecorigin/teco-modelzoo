# PyTorch

## 介绍

PyTorch是目前主流的开源的AI学习框架，主要用于构建和训练神经网络。提供了简单易用的API，使得定义模型、加载数据和训练模型变得简单。太初元碁对其进行了深度的适配和优化，新增SDAA加速卡和自研软件栈支持，并针对分类、检测、分割、自然语言处理、语音等场景常用的各类经典和前沿的AI模型适配支持。
