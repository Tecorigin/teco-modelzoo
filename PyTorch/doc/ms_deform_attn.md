难易程度：中等偏难
该算子计算逻辑清晰，没有矩阵乘法，但存在大量不规则访存和反向散射累加。要在 teco 众核芯片上高效实现，需要结合架构特点进行精细优化，整体难度中等偏上。

基础知识：
1. 芯片基础架构与存储层次
2. DMA 数据搬运，SIMD 向量计算方法
3. 异步操作并行计算等相关知识，包括但不限于：核心间同步、异步 DMA 发射与等待、双缓冲等技术，使数据搬运和计算尽可能重叠，最大化核心利用率

指导建议：
1. 充分理解算法逻辑，用 Python（NumPy）或 C 语言实现完整的逐元素前向与反向逻辑，确保数值正确性。这个串行版本既可以加深对算法的理解，也能作为后续众核版本的“黄金标准”进行精度对比。
2. 分析算法细节，合理设计任务的粒度和负载均衡，充分发挥众核架构多核心的优势进行并行操作
3. 充分利用 SPM 优势将数据尽可能多的重复利用，减少主存数据搬运
4. 根据情况可考虑是否需要额外的主存来存储中间计算结果，或者对部分数据进行重计算
